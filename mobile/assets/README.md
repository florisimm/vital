# Assets

Placeholder directory. Add real artwork before building:

- `icon.png` — 1024×1024 app icon
- `adaptive-icon.png` — 1024×1024 Android adaptive foreground
- `splash.png` — splash image (dark `#050608` background)

These are referenced by `app.json`. Until they exist, `expo start` works but
`expo prebuild` / EAS Build will complain.
